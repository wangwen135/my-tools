# Windows 批处理实用工具

本目录包含一系列实用的Windows批处理脚本，用于简化日常操作。

## 脚本列表及功能说明

### checksum.bat
计算文件的MD5、SHA1和SHA256哈希值。
使用方法：`checksum [文件名]`

### clear.bat
清除命令提示符窗口的屏幕内容，相当于Linux的clear命令。
使用方法：直接运行 `clear`

### exp.bat
使用Windows资源管理器打开当前目录。
使用方法：直接运行 `exp`

### grep.bat
模拟Linux grep命令，在文件中搜索字符串（忽略大小写）。
使用方法：`grep [搜索字符串]`

### ll.bat
显示目录中的文件和子目录详情，相当于`dir`命令。
使用方法：直接运行 `ll`

### ls.bat
以宽列表格式显示目录中的文件和子目录，相当于`dir /W`命令。
使用方法：直接运行 `ls`

### pwd.bat
显示当前工作目录的完整路径，相当于`chdir`命令。
使用方法：直接运行 `pwd`

### sp.bat
快速打开Windows截图工具（Snipping Tool）。
使用方法：直接运行 `sp`

### 关机.bat
先强制关闭有道云笔记(YoudaoNote.exe)进程，然后在6秒后关机。
使用方法：直接运行 `关机`

## 使用方法

1. 将本目录复制到您希望的位置，例如 `D:\tools\bat`
2. 将该目录添加到Windows的PATH环境变量中：
   - 右键点击"此电脑"，选择"属性"
   - 点击"高级系统设置"
   - 点击"环境变量"
   - 在"系统变量"中找到"Path"，点击"编辑"
   - 点击"新建"，输入批处理脚本所在目录的完整路径
   - 点击"确定"保存更改
3. 现在您可以在任意命令提示符窗口中直接运行这些脚本了！
